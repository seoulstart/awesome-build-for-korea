# Sumber Daya untuk Membangun Produk di Korea

> Semua yang Anda butuhkan untuk merilis produk ke pasar Korea, dikurasi untuk developer asing dan builder yang dibantu AI.

Daftar terkurasi berisi sumber daya pembayaran, verifikasi identitas, peta, perpesanan, AI, dan data pemerintah untuk merilis produk di Korea, masing-masing dianotasi berdasarkan seberapa bisa dipakai dari luar negeri. Dikelola oleh [Seoulstart](https://seoulstart.com/?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=id).

> **Catatan**: Versi bahasa Indonesia ini adalah ringkasan. Daftar lengkap, dengan setiap sumber daya dan tingkat akses untuk orang asing (🛂 ✅ bisa dipakai dari luar negeri / ⚠️ ada hambatan / ❌ butuh entitas atau domisili Korea), ada di [README bahasa Inggris](README.md).

## Paket awal tanpa hambatan

Bisa dipakai dari luar negeri dengan key gratis, tanpa badan usaha atau domisili Korea:

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Pembayaran Korea (Toss, KakaoPay, NaverPay) lewat satu endpoint MCP resmi.
- [Upstage Solar](https://www.upstage.ai) - LLM dwibahasa Korea/Inggris, kompatibel dengan OpenAI, pendaftaran global.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth untuk ~47 juta pengguna; dokumentasi bahasa Inggris; menerima nomor telepon asing.
- [Portal Data Publik (data.go.kr)](https://www.data.go.kr/en/index.do) - Lebih dari 70.000 dataset pemerintah; portal bahasa Inggris, daftar via email.
- [Stripe (metode Korea)](https://docs.stripe.com/payments/countries/korea) - Kartu dan dompet Korea tanpa entitas Korea.
