# Risorse per Costruire in Corea

> Tutto ciò che serve per lanciare un prodotto sul mercato coreano, selezionato per sviluppatori stranieri e builder assistiti dall'IA.

Un elenco curato di risorse per pagamenti, verifica dell'identità, mappe, messaggistica, IA e dati pubblici per lanciare un prodotto in Corea, ognuna annotata in base a quanto è utilizzabile dall'estero. Mantenuto da [Seoulstart](https://seoulstart.com/?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=it).

> **Nota**: questa versione italiana è un riassunto. L'elenco completo, con ogni risorsa e il suo livello di accesso per gli stranieri (🛂 ✅ utilizzabile dall'estero / ⚠️ con ostacoli / ❌ richiede entità o residenza coreana), è nel [README in inglese](README.md).

## Kit di avvio senza attriti

Utilizzabili dall'estero con una chiave gratuita, senza azienda o residenza coreana:

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Pagamenti coreani (Toss, KakaoPay, NaverPay) tramite un unico endpoint MCP ufficiale.
- [Upstage Solar](https://www.upstage.ai) - LLM bilingue coreano/inglese, compatibile con OpenAI, registrazione globale.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth per ~47 milioni di utenti; documentazione in inglese; accetta numeri di telefono stranieri.
- [Portale dei dati pubblici (data.go.kr)](https://www.data.go.kr/en/index.do) - Oltre 70.000 dataset governativi; portale in inglese, registrazione via e-mail.
- [Stripe (metodi Corea)](https://docs.stripe.com/payments/countries/korea) - Carte e wallet coreani senza un'entità coreana.
