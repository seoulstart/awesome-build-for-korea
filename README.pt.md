# Recursos para Construir na Coreia

> Tudo o que você precisa para lançar um produto no mercado coreano, com curadoria para desenvolvedores estrangeiros e builders assistidos por IA.

Uma lista com curadoria de recursos de pagamentos, verificação de identidade, mapas, mensageria, IA e dados públicos para lançar um produto na Coreia, cada um anotado conforme o quanto é utilizável de fora do país. Mantida pela [Seoulstart](https://seoulstart.com/?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=pt).

> **Nota**: Esta versão em português é um resumo. A lista completa, com cada recurso e seu nível de acesso para estrangeiros (🛂 ✅ utilizável de fora / ⚠️ com obstáculos / ❌ exige entidade ou residência coreana), está no [README em inglês](README.md).

## Kit inicial sem fricção

Funcionam de fora da Coreia com uma chave gratuita e sem empresa ou residência coreana:

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Pagamentos coreanos (Toss, KakaoPay, NaverPay) por um único endpoint MCP oficial.
- [Upstage Solar](https://www.upstage.ai) - LLM bilíngue coreano/inglês, compatível com OpenAI, cadastro global.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth para ~47 milhões de usuários; documentação em inglês; aceita telefone estrangeiro.
- [Portal de Dados Públicos (data.go.kr)](https://www.data.go.kr/en/index.do) - Mais de 70.000 conjuntos de dados do governo; portal em inglês, cadastro por e-mail.
- [Stripe (métodos da Coreia)](https://docs.stripe.com/payments/countries/korea) - Cartões e carteiras coreanas sem uma entidade coreana.
