# Sumber untuk Membina Produk bagi Pasaran Korea

> Segala yang anda perlukan untuk melancarkan produk ke pasaran Korea, dikurasi untuk pembangun asing dan builder dibantu AI.

Senarai terkurasi sumber pembayaran, pengesahan identiti, peta, pemesejan, AI dan data kerajaan untuk melancarkan produk di Korea, setiap satu dianotasi mengikut sejauh mana ia boleh digunakan dari luar negara. Diselenggara oleh [Seoulstart](https://seoulstart.com/?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=ms).

> **Nota**: Versi bahasa Melayu ini ialah ringkasan. Senarai penuh, dengan setiap sumber dan tahap aksesnya untuk orang asing (🛂 ✅ boleh guna dari luar negara / ⚠️ ada halangan / ❌ perlu entiti atau kediaman Korea), ada dalam [README bahasa Inggeris](README.md).

## Pek permulaan tanpa halangan

Boleh digunakan dari luar negara dengan key percuma, tanpa perniagaan atau kediaman Korea:

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Pembayaran Korea (Toss, KakaoPay, NaverPay) melalui satu endpoint MCP rasmi.
- [Upstage Solar](https://www.upstage.ai) - LLM dwibahasa Korea/Inggeris, serasi OpenAI, pendaftaran global.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth untuk ~47 juta pengguna; dokumentasi bahasa Inggeris; menerima nombor telefon asing.
- [Portal Data Awam (data.go.kr)](https://www.data.go.kr/en/index.do) - Lebih 70,000 set data kerajaan; portal bahasa Inggeris, daftar melalui e-mel.
- [Stripe (kaedah Korea)](https://docs.stripe.com/payments/countries/korea) - Kad dan dompet Korea tanpa entiti Korea.
