# Ресурсы для запуска продукта в Корее

> Всё необходимое, чтобы вывести продукт на корейский рынок, собрано для иностранных разработчиков и билдеров, работающих с ИИ.

Кураторская подборка ресурсов по платежам, проверке личности, картам, мессенджингу, ИИ и государственным данным для запуска продукта в Корее. Каждый помечен по тому, насколько им можно пользоваться из-за рубежа. Поддерживается [Seoulstart](https://seoulstart.com/ru?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=ru).

> **Примечание**: русская версия — это краткое изложение. Полный список со всеми ресурсами и пометками о доступе для иностранцев (🛂 ✅ доступно из-за рубежа / ⚠️ есть препятствия / ❌ нужна корейская компания или вид на жительство) — в [README на английском](README.md).

## Стартовый набор без барьеров

Работают из-за рубежа с бесплатным ключом, без корейской компании или резидентства:

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Корейские платежи (Toss, KakaoPay, NaverPay) через единый официальный MCP-эндпоинт.
- [Upstage Solar](https://www.upstage.ai) - Двуязычная корейско-английская LLM, совместима с OpenAI, регистрация по всему миру.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth для ~47 млн пользователей; документация на английском; принимает иностранные телефоны.
- [Портал открытых данных (data.go.kr)](https://www.data.go.kr/en/index.do) - Более 70 000 государственных датасетов; портал на английском, регистрация по почте.
- [Stripe (методы Кореи)](https://docs.stripe.com/payments/countries/korea) - Корейские карты и кошельки без корейской компании.
