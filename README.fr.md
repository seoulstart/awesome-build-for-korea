# Ressources pour Lancer un Produit en Corée

> Tout ce qu'il faut pour lancer un produit sur le marché coréen, sélectionné pour les développeurs étrangers et les builders assistés par IA.

Une liste sélectionnée de ressources de paiement, de vérification d'identité, de cartographie, de messagerie, d'IA et de données publiques pour lancer un produit en Corée, chacune annotée selon son utilisabilité depuis l'étranger. Maintenue par [Seoulstart](https://seoulstart.com/?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=fr).

> **Note** : cette version française est un résumé. La liste complète, avec chaque ressource et son niveau d'accès pour les étrangers (🛂 ✅ utilisable depuis l'étranger / ⚠️ avec obstacles / ❌ entité ou résidence coréenne requise), se trouve dans le [README en anglais](README.md).

## Kit de démarrage sans friction

Utilisables depuis l'étranger avec une clé gratuite, sans entreprise ni résidence coréenne :

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Paiements coréens (Toss, KakaoPay, NaverPay) via un seul endpoint MCP officiel.
- [Upstage Solar](https://www.upstage.ai) - LLM bilingue coréen/anglais, compatible OpenAI, inscription mondiale.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth pour ~47 millions d'utilisateurs ; documentation en anglais ; téléphone étranger accepté.
- [Portail de données publiques (data.go.kr)](https://www.data.go.kr/en/index.do) - Plus de 70 000 jeux de données gouvernementaux ; portail en anglais, inscription par e-mail.
- [Stripe (méthodes Corée)](https://docs.stripe.com/payments/countries/korea) - Cartes et portefeuilles coréens sans entité coréenne.
