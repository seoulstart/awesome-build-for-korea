# Recursos para Construir en Corea

> Todo lo que necesitas para lanzar un producto en el mercado coreano, curado para desarrolladores extranjeros y builders asistidos por IA.

Una lista curada de recursos de pagos, verificación de identidad, mapas, mensajería, IA y datos públicos para lanzar un producto en Corea, cada uno anotado según lo usable que es desde fuera del país. Mantenida por [Seoulstart](https://seoulstart.com/?utm_source=github&utm_medium=referral&utm_campaign=awesome-build-for-korea&utm_content=es).

> **Nota**: Esta versión en español es un resumen. La lista completa, con cada recurso y su nivel de acceso para extranjeros (🛂 ✅ usable desde fuera / ⚠️ con obstáculos / ❌ requiere entidad o residencia coreana), está en el [README en inglés](README.md).

## Kit de inicio sin fricción

Funcionan desde fuera de Corea con una clave gratuita y sin empresa ni residencia coreana:

- [PortOne MCP](https://github.com/portone-io/mcp-server) - Pagos coreanos (Toss, KakaoPay, NaverPay) a través de un único endpoint MCP oficial.
- [Upstage Solar](https://www.upstage.ai) - LLM bilingüe coreano/inglés, compatible con OpenAI, registro global.
- [Kakao Login](https://developers.kakao.com/docs/latest/en/kakaologin/rest-api) - OAuth para ~47 millones de usuarios; documentación en inglés; acepta teléfonos extranjeros.
- [Portal de Datos Públicos (data.go.kr)](https://www.data.go.kr/en/index.do) - Más de 70.000 conjuntos de datos del gobierno; portal en inglés, registro por correo.
- [Stripe (métodos de Corea)](https://docs.stripe.com/payments/countries/korea) - Tarjetas y monederos coreanos sin una entidad coreana.
